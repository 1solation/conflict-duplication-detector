Password Policy

Passwords must contain a minimum of 12 characters with at least one uppercase letter, one lowercase letter, one number, and one special character. Password rotation is required every 60 days.

Access Management

All access requests must be submitted via the ServiceNow portal. Standard access is provisioned within 48 hours. Access logs are reviewed quarterly.

Data Protection

All confidential data must be encrypted using AES-256. Sensitive information should not be stored on personal devices or removable media.
